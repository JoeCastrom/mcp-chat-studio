codebeamer_query_items

Query tracker items using CBQL (Codebeamer Query Language). Returns a paginated list of items matching the query.

Parameters:

NameTypeRequiredDescription
queryString string Yes CBQL query string. Example: "item.tracker.id IN (12345) AND item.status = 'Open'"
page number No Page number starting from 1
pageSize number No Number of items per page (max: 500)

Example:

{
  "tool": "codebeamer_query_items",
  "arguments": {
    "queryString": "example string",
    "page": 42,
    "pageSize": 42
  }
}
